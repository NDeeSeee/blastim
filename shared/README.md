# Общие ресурсы / Shared

Эта директория содержит материалы, общие для всех практикумов в курсе.

## Содержимое

| Файл | Описание | Для кого |
|------|---------|---------|
| **README.md** | Этот файл | |

## Быстрый старт

### Студентам

1. Прочитайте **COURSE_STRUCTURE.md** для понимания структуры курса
2. Выберите интересующий вас практикум:
   - `genome_annotation/` — Практикум 1: Аннотация прокариотического генома (*B. subtilis*)
   - `variant_calling/` — Практикум 2: Поиск геномных вариантов (*B. subtilis*)
   - `variant_interpretation/` — Практикум 3: Интерпретация вариантов (*H. sapiens* GRCh38)
3. Следуйте инструкциям в **README.md** выбранного практикума

### Преподавателям / Администраторам

1. Прочитайте **HPC_SETUP.md** для подготовки системы
2. Для каждого практикума см. **INSTRUCTOR.md** в его папке
3. При проблемах консультируйтесь с **TROUBLESHOOTING_COMMON.md**

## Часто задаваемые вопросы

**Q: Что будет добавлено в shared/ в будущем?**

A: По мере развития курса планируется добавить:
- Общие скрипты для управления всеми окружениями
- Больше shared утилит для валидации и отладки

**Q: Почему данные для практикумов 1–2 скачиваются каждый раз?**

A: Каждый практикум полностью независим, чтобы студенты могли брать любой из них без предварительных шагов. Размер данных B. subtilis 168 (~4 МБ) небольшой.

**Q: Практикум 3 требует интернет?**

A: Нет. Демо-данные включены в репозиторий (`variant_interpretation/data/input/`). VEP REST API используется опционально — при отсутствии интернета скрипты автоматически переключаются на предобработанный файл.

**Q: Могу ли я запустить все три практикума одновременно?**

A: Да, они полностью независимы и используют разные conda-окружения:
- Практикум 1: `ngs-annotation`
- Практикум 2: `ngs-variants`
- Практикум 3: `ngs-interpret`
