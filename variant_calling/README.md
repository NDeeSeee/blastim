# Поиск вариантов (Variant Calling)

> **Статус:** в разработке

Практикум по поиску геномных вариантов (SNP/InDel) с использованием стандартных инструментов биоинформатики на HPC.

## Планируемые темы

1. Выравнивание прочтений на референсный геном (BWA / Bowtie2)
2. Обработка BAM-файлов (SAMtools)
3. Поиск вариантов (BCFtools / GATK)
4. Фильтрация и аннотация вариантов (SnpEff / SnpSift)
